`squash-utils` is a set of standalone bash and python scripts used for desktop
theme management and manipulation. 

To see the full project, see the master `squash` repository on [github](https://github.com/JLErvin/squash)
or [gitlab](https://gitlab.com/JLErvin/squash)

## Scripts

`squash-utils` contains the following scripts:

* `reload_bar`: kill and reload lemonbar/polybar
* `reload_bspwm`: reload bspwm
* `reload_dunst`: update `dunstrc` and reload
* `reload_nvim`: reload all instances of `nvim` using `nvr-remote`
* `reload_terminal`: send escape sequences to all open terminals using update colors
* `reload_wallpaper`: set the current wallpaper
* `reload_wchef`: reload windowchef
* `zucchini.py`: create `squash` theme from terminal.sexy (created by t60r)
